# Shared Understanding Model — Categories & Minimum Thresholds (MVP)

| Category | Minimum | Description |
|---|---|---|
| Actors & Roles | 2 | who interacts + what each role can do |
| Business Rules | 3 | business rules / policies / eligibility |
| Entities & Data | 2 | key entities and main attributes |
| Constraints | 1 | technical, legal, or operational limits |
| Dependencies | 1 | external systems, other features |
| Edge Cases / Exceptions | 2 | non-standard scenarios |
| Goals / Acceptance Criteria | 1 | how we know it's solved |

Used by the Clarification Engine to decide when a category is
"sufficiently covered", and by Open Questions to detect gaps.
